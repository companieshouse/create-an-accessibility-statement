Create an accessibility statement

Use this service to create or update an accessibility statement for a Companies House digital service.

What you’ll need

To create the statement, you’ll need to provide:

  • the service name
  • relevant dates – the date it was audited, the date the statement will be published, and the date the statement was first published (if it already exists)
  • who carried out the audit
  • a short summary of what was tested
  • the audit result - pass, partial pass or fail

If the service had a partial pass or fail

You’ll also need:

  • a summary of the accessibility issues
  • to explain why the service didn’t meet accessibility requirements
  • a description of each accessibility issue
  • to select which WCAG success criterion each issue relates to

What happens next

When you’ve completed the service, you’ll have content you can use in a GOV.UK accessibility statement. This will be based on the sample accessibility statement.

It will be generated in 3 formats:

  • standard preview
  • HTML
  • markdown

You should copy and paste the statement in your chosen format immediately to avoid losing the information. You cannot save your answers in this service.
